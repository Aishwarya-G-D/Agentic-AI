# Challenges

Hands-on challenges for the Mainframe Modernization Summit 2026 Specialist Workshop. Each challenge builds on the previous one, progressively increasing in complexity and autonomy.

| # | Challenge | Duration | Skill Level | Status |
|---|-----------|----------|-------------|--------|
| 1 | [Hello World](../tutorials/hello-world.md) | 30 min | Level 1 — Follow along | Tutorial |

## Skill Levels

| Level | Description |
|-------|-------------|
| **Level 1** | Follow step-by-step instructions. Minimal deviation from the guide. |
| **Level 2** | Demonstrates GitHub, VS Code, and dev tool proficiency. Applies foundational Python code reading skills. Understands existing agent code structure, configuration, and dependencies. |
| **Level 3** | Designs and builds independently with minimal guidance. Extends existing patterns. |
