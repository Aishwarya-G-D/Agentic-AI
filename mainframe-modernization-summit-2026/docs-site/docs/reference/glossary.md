# Glossary

| Term | Meaning |
|------|---------|
| **ADK** | Agent Development Kit — IBM Watsonx Orchestrate's CLI + SDK for authoring agents and tools. |
| **Agent** | An LLM-backed assistant defined by a YAML spec, exposing tools and a behaviour. |
| **Tool** | A callable (Python or OpenAPI) the agent can invoke to take action. |
| **z/OSMF** | z/OS Management Facility — REST API surface for managing a mainframe. |
| **LPAR** | Logical Partition — an isolated slice of a physical mainframe. |
| **Connection** | Named credentials + endpoint config consumed by tools at runtime. |
| **JES2** | Job Entry Subsystem 2 — z/OS batch job scheduler. |
| **RACF** | Resource Access Control Facility — z/OS security manager. |
